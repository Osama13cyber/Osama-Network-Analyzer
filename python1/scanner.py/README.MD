# 🛡️ OSAMA NETWORK ANALYZER v2.0

**OSAMA NETWORK ANALYZER** is a professional-grade network security and discovery tool. It allows users to scan their local network, identify connected devices, detect manufacturers (Vendors), and find open ports to assess potential security risks.



## 🚀 Key Features
* **Automatic Network Detection:** Automatically finds your IP range (e.g., 192.168.1.0/24).
* **Device Discovery:** Fast ARP-based scanning to find all live devices.
* **Port Scanning:** Identifies common services like HTTP, SSH, FTP, and SMB.
* **Vendor Lookup:** Real-time identification of device manufacturers (Apple, Samsung, etc.).
* **Security Risk Rating:** Flags dangerous open ports (e.g., SMB) as "HIGH RISK".
* **Professional Dashboard:** Modern, dark-themed web interface for easy analysis.
